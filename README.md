<h1>📌 Event Attendance System</h1>

<p>
  Streamline your event management with <strong>QR code-based registration and attendance tracking</strong>.  
  Experience <strong>seamless check-ins</strong> and <strong>real-time analytics</strong>.
</p>

<p><strong>Live Demo:</strong> <a href="https://event-registration-system-4ejh.vercel.app/" target="_blank">https://event-registration-system-4ejh.vercel.app/</a></p>

---

<h2>🚀 How it Works</h2>
<p><strong>Register → Get QR Code → Scan at Event → Track Attendance → View Reports</strong></p>

---

<h2>✨ Features</h2>
<ul>
  <li>Easy Registration with instant QR code generation and email confirmations</li>
  <li>QR Code Scanning with duplicate prevention</li>
  <li>Real-time Tracking via live dashboard and notifications</li>
  <li>Analytics & Reports exportable to Excel, CSV, and JSON</li>
  <li>Admin Dashboard for event and participant management</li>
  <li>Sample Data for quick testing and demos</li>
</ul>

<h2>⚙️ Setup Instructions</h2>

<h3>1️⃣ Clone the Repository</h3>
<pre><code>git clone https://github.com/YOUR_USERNAME/event-attendance-system.git
cd event-attendance-system
</code></pre>

<h3>2️⃣ Install Dependencies</h3>
<pre><code>npm install
</code></pre>

<h3>3️⃣ Add Environment Variables in <code>.env</code></h3>
<pre><code>DATABASE_URL=your_database_url
EMAIL_SERVICE_API_KEY=your_email_service_key
QR_SECRET_KEY=your_qr_secret
</code></pre>

<h3>4️⃣ Run the Project</h3>
<pre><code>npm run dev
</code></pre>
