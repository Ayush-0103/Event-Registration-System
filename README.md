Event Attendance System

Streamline your event management with QR code-based registration and attendance tracking.
Experience seamless check-ins and real-time analytics.

Live Demo: https://event-registration-system-4ejh.vercel.app/

How it Works

Register → Get QR Code → Scan at Event → Track Attendance → View Reports

Features

Easy Registration with instant QR code generation and email confirmations

QR Code Scanning with duplicate prevention

Real-time Tracking via live dashboard and notifications

Analytics & Reports exportable to Excel, CSV, and JSON

Admin Dashboard for event and participant management

Sample Data for quick testing and demos

Deployment

This project is deployed on Vercel.
Live link: https://event-registration-system-4ejh.vercel.app/

Setup Instructions

Clone the Repository

git clone https://github.com/YOUR_USERNAME/event-attendance-system.git
cd event-attendance-system


Install Dependencies

npm install


Add Environment Variables in .env

DATABASE_URL=your_database_url
EMAIL_SERVICE_API_KEY=your_email_service_key
QR_SECRET_KEY=your_qr_secret


Run the Project

npm run dev
